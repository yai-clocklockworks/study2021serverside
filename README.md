## よく使うコマンド

- コンテナ起動
    `docker-compose up`

- コンテナ停止
    `docker-compose stop`

- コンテナ削除
    `docker-compose down`

- コンテナ一覧
    `docker ps`

- コンテナ内でコマンド実行
    `docker exec -it {コンテナID} rails -v`

- ブラウザ
    `http://localhost:3000/`
